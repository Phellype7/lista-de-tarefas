# lista-de-tarefas
Task List in Python
This is a simple Python program that allows you to add, remove, and view tasks. Tasks are saved in a .txt file, so you wonâ€™t lose them when closing the program.

How to use

Run the Python file.
Choose an option from the menu:
1: Add a task
2: Remove a task
3: View tasks
4: Exit

Tasks will be saved automatically.

Features
Add new tasks.
Prevent duplicate tasks.
Remove one or multiple tasks at once.
List tasks with numbering.
Automatically save tasks in a .txt file.

Requirements
Python 3.x
No extra libraries needed
